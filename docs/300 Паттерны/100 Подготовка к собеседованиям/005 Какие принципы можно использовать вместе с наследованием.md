В объектно-ориентированном программировании _наследование_ является одним из основных принципов, но его можно использовать совместно с другими принципами, чтобы создавать более эффективные и гибкие программы.

_`Композиция`_ - _принцип_, который позволяет создавать объекты из других объектов. Композиция _может использоваться вместе с наследованием для создания сложных объектов, которые состоят из нескольких других объектов_. Например, класс-наследник может использовать объекты родительского класса в составе своих свойств.
