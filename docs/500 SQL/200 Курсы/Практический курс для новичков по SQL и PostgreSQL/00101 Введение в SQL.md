### Введение в SQL

База данных (БД) - набор взаимосвязанных данных.

Система управления базами данных (СУБД) - комплекс программных средств для управления данными.

СУБД отвечает за:
* поддержку языка БД,
* механизмы хранения и извлечения данных,
* оптимизацию процессов извлечения данных.

Типы СУБД:
* Файл-серверные: Microsoft Access,
* Клиент-серверные: MySQL, PostgreSQL...
* Встраиваемые: SQLite.

Основные клиент-серверные СУБД:
- MySQL,
- PostgreSQL,
- Oracle,
- MS SQL,
- Maria DB.
Все вышеперечисленные СУБД - реляционные и все они поддерживают язык SQL.

Не реляционные СУБД существовали задолго до реляционных (иерархические, например).

На смену пришли реляционные (начиная с 1970х) и спустя несколько декад, не реляционные СУБД возродились (расцвет в 2010х), появилось NoSQL движение (Google big table).

mongoDB - одна из наиболее популярных NoSQL-СУБД.
Реляционные СУБД не хуже и не лучше не реляционных СУБД. 

У всех видов (SQL и NoSQL) есть свои преимущества и недостатки.
