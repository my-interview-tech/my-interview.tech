### **Код для вывода общего объема памяти и свободной памяти в гигабайтах**

```javascript
const os = require("os")

const totalMemoryGB = (os.totalmem() / 1024 ** 3).toFixed(2)
const freeMemoryGB = (os.freemem() / 1024 ** 3).toFixed(2)

console.log(`Общий объем памяти: ${totalMemoryGB} ГБ`)
console.log(`Свободная память: ${freeMemoryGB} ГБ`)
```

---

### **Описание кода**

1. **`os.totalmem()`** — возвращает общий объем оперативной памяти в байтах.
2. **`os.freemem()`** — возвращает количество свободной памяти в байтах.
3. **`/ 1024 ** 3`\*\* — перевод байты в гигабайты (1 ГБ = 1024³ байт).
4. **`.toFixed(2)`** — округляет до двух знаков после запятой.

---

### **Пример вывода** (на системе с 16 ГБ RAM)

```
Общий объем памяти: 15.89 ГБ
Свободная память: 7.25 ГБ
```
