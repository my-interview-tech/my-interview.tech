## Удаление глобально установленного npm-пакета

Чтобы удалить **глобально установленный npm-пакет**, используйте команду **`npm uninstall -g <пакет>`**.

### Шаги для удаления глобального пакета:

1. Откройте терминал или командную строку.
2. Используйте команду **`npm uninstall -g`**, указав имя пакета, который вы хотите удалить.

#### Пример:

```bash
npm uninstall -g typescript
```

Эта команда удалит глобально установленный пакет `typescript`.

### Проверка успешного удаления

Чтобы убедиться, что пакет был успешно удален, можно проверить список глобально установленных пакетов:

```bash
npm list -g --depth=0
```

### Расширенные опции

- С флагом **`--dry-run`** можно проверить, что будет удалено, без фактического удаления:

```bash
npm uninstall -g typescript --dry-run
```

- Если у вас нет прав на удаление глобальных пакетов, используйте `sudo` (Linux/Mac):

```bash
sudo npm uninstall -g typescript
```

### Примечания:

- Глобальные пакеты устанавливаются с флагом **`-g`** и обычно находятся в глобальной папке npm:
  - Linux/Mac: `/usr/local/lib/node_modules` или `~/.npm-global/lib/node_modules`
  - Windows: `%AppData%\npm\node_modules`
- Вы также можете использовать альтернативный синтаксис: `npm remove -g <пакет>` или `npm rm -g <пакет>`
