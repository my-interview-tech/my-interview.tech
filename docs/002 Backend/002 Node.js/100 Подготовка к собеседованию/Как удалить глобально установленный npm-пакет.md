Чтобы удалить **глобально установленный npm-пакет**, используйте команду **`npm uninstall -g <пакет>`**.

### Шаги для удаления глобального пакета:

1. Откройте терминал или командную строку.
2. Используйте команду **`npm uninstall -g`**, указав имя пакета, который вы хотите удалить.

#### Пример:

bash

КопироватьРедактировать

`npm uninstall -g typescript`

Эта команда удалит глобально установленный пакет `typescript`.

### Дополнительные флаги:

- Если вы хотите удостовериться, что пакет удалён, можно добавить флаг **`--save`**, хотя для глобальных пакетов это не обязательно. Просто **`-g`** достаточно.

#### Примечание:

- Глобальные пакеты устанавливаются с флагом **`-g`** и обычно находятся в глобальной папке npm, например, в `/usr/local/lib/node_modules` на Unix-подобных системах или в другой директории, в зависимости от настроек npm.
