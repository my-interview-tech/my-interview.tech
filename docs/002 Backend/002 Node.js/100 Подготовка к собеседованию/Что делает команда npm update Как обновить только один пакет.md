**npm update** - это команда для обновления пакетов в проекте Node.js до их последних доступных версий, соответствующих ограничениям, указанным в package.json.

### Что делает команда npm update

1. **Проверяет обновления** - находит все пакеты, для которых доступны более новые версии
2. **Соблюдает семантическое версионирование** - обновляет только до тех версий, которые совместимы с указанными в package.json
3. **Обновляет package.json** - если используется флаг `--save`
4. **Обновляет package-lock.json** - отражает точные версии всех установленных пакетов

По умолчанию `npm update` обновляет только до версий, соответствующих указанным диапазонам в package.json. Например, для зависимости `"express": "^4.17.1"` будут установлены обновления в пределах мажорной версии 4.

### Как обновить только один пакет

Чтобы обновить только один конкретный пакет, нужно указать его имя:

```bash
npm update <имя-пакета>
```

Например:

```bash
npm update express
```

### Дополнительные полезные флаги

1. **Обновить с сохранением версии в package.json**:

```bash
npm update express --save
```

2. **Обновить глобально установленный пакет**:

```bash
npm update -g <имя-пакета>
```

3. **Обновить до последней мажорной версии** (может нарушить совместимость):

```bash
npm update <имя-пакета> --force
```

### Проверка доступных обновлений без их установки

Чтобы просто проверить, какие пакеты можно обновить:

```bash
npm outdated
```

Вывод будет цветным, где:

- **Красный** - доступно обновление мажорной версии
- **Желтый** - доступно обновление в рамках указанного диапазона версий
- **Зеленый** - пакет установлен с последней версией из указанного диапазона
