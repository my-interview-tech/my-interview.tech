В **Node.js REPL** можно использовать символ **`_`** для доступа к результату предыдущего выражения. Вот как можно создать цикл, который будет выводить числа от 1 до 5, используя **`_`**:

1. В REPL, введите следующий код:
    
    javascript
    
    КопироватьРедактировать
    
    `> let i = 1; > while (i <= 5) { ...   console.log(i); ...   i++; ... }`
    

Но, поскольку REPL позволяет использовать **`_`** для получения результата предыдущего выражения, можно сделать вывод чисел с использованием этого механизма:

2. Теперь используйте **`_`** для отображения текущего числа:
    
    javascript
    
    КопироватьРедактировать
    
    `> _ = 1;  > while (_ <= 5) { console.log(_); _++; }`
    

Этот цикл выведет:

КопироватьРедактировать

`1 2 3 4 5`

### Объяснение:

- В данном примере **`_`** используется для хранения текущего значения, начиная с 1, и увеличивается на каждом шаге.
- **`console.log(_)`** выводит текущее значение **`_`** в REPL, которое затем увеличивается в цикле.
