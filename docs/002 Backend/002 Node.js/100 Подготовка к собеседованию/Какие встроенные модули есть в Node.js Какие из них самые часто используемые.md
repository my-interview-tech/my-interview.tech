Node.js включает в себя множество встроенных (core) модулей, которые обеспечивают работу с файловой системой, сетью, потоками и другими функциями без необходимости установки дополнительных пакетов.

**Основные встроенные модули в Node.js:**

1. **fs (File System)** – работа с файловой системой (чтение, запись, удаление файлов).
2. **http / https** – создание веб-серверов и работа с HTTP(S)-запросами.
3. **url** – разбор и формирование URL.
4. **path** – работа с путями к файлам и директориям.
5. **os** – получение информации об операционной системе.
6. **events** – создание и управление событиями через `EventEmitter`.
7. **crypto** – шифрование, хеширование, работа с алгоритмами безопасности.
8. **util** – вспомогательные утилиты (например, `promisify` для работы с колбэками в виде промисов).
9. **stream** – работа с потоками данных (чтение/запись в потоковом режиме).
10. **child_process** – запуск дочерних процессов (например, выполнение команд оболочки).

### **Самые часто используемые модули:**

1. **fs** – чтение и запись файлов (`fs.readFile`, `fs.writeFile`).
2. **http** – создание HTTP-серверов (`http.createServer()`).
3. **path** – манипуляции с путями (`path.join()`, `path.resolve()`).
4. **events** – управление событиями (`EventEmitter`).
5. **crypto** – генерация хэшей, работа с токенами (например, `crypto.createHash('sha256')`).

Эти модули встроены в Node.js, поэтому их можно использовать без установки через `require('module_name')`.
