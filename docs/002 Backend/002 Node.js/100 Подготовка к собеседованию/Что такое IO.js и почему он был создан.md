**IO.js** – это форк Node.js, созданный в **2014 году** группой разработчиков, недовольных медленным развитием Node.js под управлением компании **Joyent**.

**Причины создания IO.js:**

1. **Медленные обновления Node.js** – в то время релизы выходили редко, а поддержка новых функций JavaScript отставала.
2. **Закрытая разработка** – сообщество не имело достаточного влияния на развитие Node.js.
3. **Отсутствие актуального V8** – IO.js быстро адаптировал новые версии движка **V8**, обеспечивая поддержку современных возможностей JS.
4. **Открытое управление** – разработка IO.js велась открыто, через **модель Open Governance**.

**Что стало с IO.js?**

В **2015 году** проект **IO.js воссоединился с Node.js**, и на его основе была создана **Node.js Foundation** (ныне OpenJS Foundation). В результате:

- Node.js получил **актуальный V8**,
- Улучшилось управление проектом,
- Начались регулярные обновления.

Сегодня IO.js больше не существует, но его идеи помогли сделать Node.js современным и открытым.
