Для импорта модуля из соседнего файла в **Node.js** есть несколько вариантов в зависимости от того, используете ли вы **CommonJS** или **ES Modules**. Рассмотрим оба варианта.

### 1. **CommonJS (используя `require()`)**

В **CommonJS** вы используете функцию `require()` для импорта модулей. В случае соседнего файла, важно указать относительный путь к файлу, используя `./` для текущей директории.

#### Пример:

Допустим, у вас есть два файла:

- `module.js`
- `main.js`

`// module.js module.exports = function greet() {   console.log('Hello from the module!'); };`

Теперь в файле `main.js` вы можете импортировать модуль с помощью `require()`:

javascript

КопироватьРедактировать

`// main.js const greet = require('./module');  // ./ указывает на текущую директорию greet();`

### 2. **ES Modules (используя `import`)**

Если вы используете **ES Modules**, то для импорта используется синтаксис `import`. Для работы с ES Modules в Node.js нужно либо использовать расширение `.mjs` для файлов, либо указать `"type": "module"` в `package.json`.

#### Пример:

Если ваши файлы также называются `module.js` и `main.js`, вы можете сделать следующее:

1. Убедитесь, что в `package.json` указан `"type": "module"`, чтобы Node.js знал, что файлы используют ES Modules:

json

КопироватьРедактировать

`{   "type": "module" }`

2. Используйте синтаксис `import` в `main.js` для импорта:

javascript

КопироватьРедактировать

`// module.js export function greet() {   console.log('Hello from the module!'); }`

javascript

КопироватьРедактировать

`// main.js import { greet } from './module.js';  // ./ указывает на текущую директорию greet();`

### Обратите внимание:

- В **CommonJS** используется `require()` и не требуется указывать расширение файла (если это `.js`).
- В **ES Modules** нужно указывать расширение файла (например, `.js` или `.mjs`).

### Заключение:

- **CommonJS**: `const module = require('./module')`
- **ES Modules**: `import { module } from './module.js'`

Выбор зависит от того, какой формат модулей используется в вашем проекте (CommonJS или ES Modules).
