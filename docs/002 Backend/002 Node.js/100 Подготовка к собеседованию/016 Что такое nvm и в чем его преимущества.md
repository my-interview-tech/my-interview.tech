**nvm** (Node Version Manager) — это утилита командной строки, которая позволяет управлять несколькими версиями Node.js на одной машине.

#### Основные возможности nvm

- **Установка разных версий Node.js**: возможность устанавливать конкретные версии без затрагивания системного Node.js
- **Быстрое переключение между версиями**: мгновенная смена версии Node.js для разных проектов
- **Использование определенной версии для проекта**: автоматическое переключение версий при смене каталогов с помощью `.nvmrc`
- **Изолированные глобальные пакеты**: каждая версия Node.js имеет свой собственный набор глобальных пакетов

#### Примеры использования nvm

**Установка nvm** (для Unix-подобных систем):

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
```

**Установка определенной версии Node.js**:
```bash
nvm install 18.16.1
```

**Использование конкретной версии**:
```bash
nvm use 16.20.0
```

**Установка и использование последней LTS-версии**:
```bash
nvm install --lts
nvm use --lts
```

**Просмотр всех установленных версий**:
```bash
nvm ls
```

**Настройка версии Node.js по умолчанию**:
```bash
nvm alias default 18.16.1
```

#### Преимущества использования nvm

1. **Совместимость проектов**: можно легко поддерживать проекты, которые требуют разных версий Node.js
2. **Тестирование в разных средах**: упрощает проверку работы кода на разных версиях Node.js
3. **Автоматизация с помощью .nvmrc**: можно создать файл `.nvmrc` с указанием требуемой версии, и при входе в директорию проекта nvm автоматически переключится на нужную версию Node.js
4. **Изоляция глобальных пакетов**: предотвращает конфликты между пакетами, установленными для разных версий Node.js

#### Пример содержимого файла .nvmrc

```
18.16.1
```

Достаточно положить такой файл в корень проекта и выполнить `nvm use` в этой директории, чтобы автоматически переключиться на указанную версию.

#### Альтернативы nvm

- **n** - более простой менеджер версий Node.js
- **nodenv** - легковесный менеджер версий
- **nodist** - менеджер версий для Windows
- **volta** - современный инструмент для управления JavaScript-инструментарием
