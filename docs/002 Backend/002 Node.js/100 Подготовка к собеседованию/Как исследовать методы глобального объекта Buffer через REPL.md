Чтобы исследовать методы глобального объекта **`Buffer`** в **Node.js REPL**, можно просто использовать доступные функции и свойства объекта **`Buffer`**, так как он является глобальным объектом в Node.js. Это позволяет вам работать с ним напрямую, без необходимости импортировать его.

### 1. Откройте REPL:

Запустите REPL командой:

bash

КопироватьРедактировать

`node`

### 2. Исследуйте методы объекта **`Buffer`**:

- Вы можете начать с того, чтобы вывести сам объект **`Buffer`**, чтобы увидеть его методы и свойства:

javascript

КопироватьРедактировать

`> Buffer [Function: Buffer]`

- Чтобы увидеть все доступные методы и свойства объекта **`Buffer`**, можно использовать команду **`console.dir`** с параметром `{showHidden: true}`:

javascript

КопироватьРедактировать

`> console.dir(Buffer, {showHidden: true, depth: null})`

Это отобразит все методы и свойства **`Buffer`** в объекте.

### 3. Пример использования методов **`Buffer`**:

- Например, вы можете создать буфер с помощью **`Buffer.alloc()`** и исследовать его методы:

javascript

КопироватьРедактировать

`> const buf = Buffer.alloc(10) undefined > buf <Buffer 00 00 00 00 00 00 00 00 00 00> > buf.length 10 > buf.toString('utf-8') '����������'`

- Также вы можете использовать методы, такие как **`Buffer.from()`**, **`Buffer.isBuffer()`**, **`Buffer.concat()`**:

javascript

КопироватьРедактировать

`> const buf1 = Buffer.from('Hello') undefined > const buf2 = Buffer.from(' World') undefined > Buffer.concat([buf1, buf2]) <Buffer 48 65 6c 6c 6f 20 57 6f 72 6c 64> > buf1.toString() + buf2.toString() 'Hello World'`

### 4. Просмотр документации:

Если вы хотите узнать подробности о каждом методе, можно обратиться к официальной документации Node.js, так как REPL не предоставляет прямого способа вывести описание метода. Например, посмотрите раздел Buffer в документации Node.js.

### 5. Использование **`Buffer.prototype`**:

Если вас интересует информация о конкретных методах, вы можете использовать **`Buffer.prototype`**:

javascript

КопироватьРедактировать

`> console.dir(Buffer.prototype, {showHidden: true, depth: null})`

Таким образом, вы можете исследовать все доступные методы для работы с объектом **`Buffer`** и их функциональность через REPL.
