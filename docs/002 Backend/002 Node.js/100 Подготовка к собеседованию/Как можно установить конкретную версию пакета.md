Чтобы установить **конкретную версию пакета** в **Node.js** с помощью **npm**, нужно указать номер версии при установке. Вот несколько способов это сделать:

### 1. **Установка конкретной версии пакета**

Чтобы установить конкретную версию пакета, укажите номер версии после имени пакета. Например, чтобы установить версию `4.17.21` пакета `lodash`, используйте следующую команду:

bash

КопироватьРедактировать

`npm install lodash@4.17.21`

### 2. **Установка конкретной версии пакета как зависимости**

Если вы хотите установить пакет с конкретной версией как зависимость в вашем проекте, добавьте флаг `--save` (для старых версий npm, начиная с версии 5 по умолчанию зависимости сохраняются в `package.json`):

bash

КопироватьРедактировать

`npm install lodash@4.17.21 --save`

Для **dev-зависимостей** используйте флаг `--save-dev`:

bash

КопироватьРедактировать

`npm install lodash@4.17.21 --save-dev`

### 3. **Установка версии в `package.json`**

Если вам нужно указать конкретную версию пакета в **`package.json`**, откройте файл и вручную измените секцию `dependencies` или `devDependencies`, указав нужную версию. Например:

json

КопироватьРедактировать

`"dependencies": {   "lodash": "4.17.21" }`

После этого выполните команду:

bash

КопироватьРедактировать

`npm install`

### 4. **Установка версии, подходящей под диапазон**

Если вы хотите установить версию пакета, подходящую под диапазон версий, можно использовать такие операторы, как:

- **`^`** — устанавливает последнюю минорную версию, совместимую с текущей.
- **`~`** — устанавливает последнюю патч-версию, совместимую с текущей.

Пример:

bash

КопироватьРедактировать

`npm install lodash@^4.17.0   # Установит последнюю версию 4.x.x npm install lodash@~4.17.0   # Установит последнюю версию 4.17.x`

### 5. **Установка последней версии пакета**

Если нужно установить **последнюю доступную версию** пакета (независимо от того, что указано в `package.json`), используйте команду:

bash

КопироватьРедактировать

`npm install lodash@latest`

### Заключение:

- Чтобы установить **конкретную версию пакета**, используйте команду `npm install <пакет>@<версия>`.
- Для установки пакета в зависимости используйте `--save` или `--save-dev` флаги.
- Также можно установить пакеты с диапазонами версий с помощью символов `^` или `~`.
