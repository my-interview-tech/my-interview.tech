Чтобы узнать версию конкретного установленного пакета в **Node.js**, можно использовать несколько методов. Вот основные способы:

### 1. **Использование команды `npm list`**

Для того чтобы узнать версию установленного пакета, можно использовать команду `npm list`, указав имя пакета.

#### Пример:

bash

КопироватьРедактировать

`npm list <пакет>`

Пример для `lodash`:

bash

КопироватьРедактировать

`npm list lodash`

Этот вывод покажет точную версию пакета, установленного в вашем проекте:

bash

КопироватьРедактировать

`my-project@1.0.0 ├── lodash@4.17.21`

Если вы хотите ограничить вывод только этой версией, можно использовать опцию `--depth=0`:

bash

КопироватьРедактировать

`npm list lodash --depth=0`

### 2. **Просмотр в `package.json`**

Если вы установили пакет как зависимость, его версия будет указана в вашем файле **`package.json`** в секции `dependencies` или `devDependencies`.

Пример:

json

КопироватьРедактировать

`"dependencies": {   "lodash": "^4.17.21" }`

### 3. **Использование команды `npm show`**

Если вы хотите узнать версию пакета, который установлен в вашем проекте, или доступную последнюю версию, используйте команду `npm show`.

#### Пример:

bash

КопироватьРедактировать

`npm show <пакет> version`

Пример для `lodash`:

bash

КопироватьРедактировать

`npm show lodash version`

Это вернет текущую доступную версию пакета из реестра npm.

### 4. **Просмотр директории `node_modules`**

Также можно посмотреть версию конкретного пакета, заглянув в папку **`node_modules`**. Например, для пакета `lodash`:

bash

КопироватьРедактировать

`cat node_modules/lodash/package.json | grep version`

Это покажет строку с версией пакета, например:

bash

КопироватьРедактировать

`"version": "4.17.21"`

### 5. **Использование `npx` (для глобальных пакетов)**

Если вы хотите узнать версию глобально установленного пакета, используйте команду с `npx`.

#### Пример:

bash

КопироватьРедактировать

`npx <пакет> --version`

Пример для `typescript`:

bash

КопироватьРедактировать

`npx typescript --version`

### Заключение:

- **`npm list <пакет>`** — покажет версию установленного пакета в проекте.
- **`npm show <пакет> version`** — покажет доступную версию пакета в реестре npm.
- **Просмотр в `package.json`** — можно увидеть версию пакета, указанную как зависимость в проекте.
