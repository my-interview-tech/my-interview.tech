## Что такое REPL в Node.js

REPL (Read-Eval-Print Loop) – это интерактивная среда Node.js, которая позволяет вводить JavaScript-код и немедленно видеть результат его выполнения.

- **Read**: Считывает ввод пользователя
- **Eval**: Оценивает введенный код
- **Print**: Выводит результат выполнения
- **Loop**: Повторяет процесс

## Базовый запуск REPL

Для запуска Node.js в режиме REPL, просто выполните в командной строке:

```bash
node
```

После этого вы увидите приглашение командной строки `>`, где можно вводить JavaScript-код.

## Пример использования

```javascript
> console.log("Hello, Node.js!")
Hello, Node.js!
undefined
> 2 + 2
4
> const obj = { name: "Node", version: "18.x" }
undefined
> obj
{ name: 'Node', version: '18.x' }
```

## Специальные команды в REPL

В режиме REPL доступны специальные команды, которые начинаются с точки:

- `.help` – показывает список доступных команд
- `.exit` или дважды нажатие Ctrl+C – выход из REPL
- `.clear` – очистка контекста выполнения
- `.save filename` – сохранение сессии в файл
- `.load filename` – загрузка JavaScript-файла в текущую сессию
- `.editor` – переход в многострочный режим редактирования

## Запуск REPL с дополнительными флагами

Node.js REPL можно запускать с различными флагами:

### Включение поддержки await на верхнем уровне

```bash
node --experimental-repl-await
```

Пример использования:

```javascript
> await Promise.resolve('Поддержка await!')
'Поддержка await!'
```

### Запуск в режиме отладки

```bash
node --inspect
```

### Загрузка модуля при запуске

```bash
node -r module-name
```

## Доступ к последнему результату

В REPL можно обращаться к результату последней операции через специальную переменную `_`:

```javascript
> 5 + 5
10
> _ * 2
20
```

## Многострочный ввод

REPL автоматически обнаруживает незавершенные выражения и позволяет продолжать ввод на следующих строках:

```javascript
> function add(a, b) {
... return a + b;
... }
undefined
> add(2, 3)
5
```

## Автодополнение кода

В REPL работает автодополнение кода с помощью клавиши Tab:

```javascript
> process. [нажатие Tab]
// Отобразится список всех доступных свойств и методов объекта process
```

## Практическое применение REPL

- Быстрое тестирование кода
- Эксперименты с API Node.js
- Изучение новых модулей и библиотек
- Отладка небольших фрагментов кода
- Выполнение простых вычислений
