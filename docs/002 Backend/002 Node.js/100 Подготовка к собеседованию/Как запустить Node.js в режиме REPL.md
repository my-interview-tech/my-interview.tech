Для запуска **Node.js** в режиме **REPL (Read-Eval-Print Loop)**, просто выполните команду:

bash

КопироватьРедактировать

`node`

Это откроет интерактивную командную строку, в которой можно выполнять JavaScript код в реальном времени.

### Что такое REPL?

- **Read**: Вводит строку (или команду) пользователя.
- **Eval**: Оценивает введенную строку как код JavaScript.
- **Print**: Выводит результат выполнения кода.
- **Loop**: Повторяет процесс для следующей строки кода.

### Пример использования:

1. Откройте терминал.
2. Выполните команду `node`.
3. Введите любой JavaScript код, например:

javascript

КопироватьРедактировать

`> console.log("Hello, Node.js!"); Hello, Node.js! undefined > 2 + 2 4`

### Особенности REPL в Node.js:

- Вы можете выполнять любые выражения и операторы JavaScript.
- Объекты, функции и модули Node.js также доступны в REPL.
- Для выхода из REPL нажмите `Ctrl + C` дважды или используйте команду `.exit`.

### Использование REPL с флагами:

Node.js REPL можно запускать с дополнительными флагами. Например:

- **`--inspect`**: Запуск REPL в режиме отладки.
    
    bash
    
    КопироватьРедактировать
    
    `node --inspect`
    
- **`--experimental-repl-await`**: Включение возможности использовать `await` в REPL.
    
    bash
    
    КопироватьРедактировать
    
    `node --experimental-repl-await`
    

REPL полезен для быстрого тестирования и экспериментов с кодом без необходимости создавать отдельные файлы.
