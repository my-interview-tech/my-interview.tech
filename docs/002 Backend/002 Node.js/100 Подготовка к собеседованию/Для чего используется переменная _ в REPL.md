# Для чего используется переменная \_ в REPL

В **Node.js REPL** (Read-Eval-Print Loop) переменная **`_`** является специальной переменной, которая используется для хранения **результата последнего вычисленного выражения**. Это позволяет удобно работать с результатами предыдущих операций без необходимости их сохранения в отдельные переменные.

## Основное назначение

Переменная **`_`** автоматически обновляется после каждого вычисления в REPL и предоставляет быстрый доступ к последнему результату. Это особенно полезно при:

- Быстром тестировании выражений
- Цепочках вычислений
- Проверке свойств объектов
- Отладке JavaScript-кода

## Примеры использования

### 1. Базовые арифметические операции

```javascript
> 2 + 3
5
> _ * 2
10
> _ - 5
5
```

Здесь сначала **`_`** содержит результат `2 + 3` (значение `5`), затем **`_`** умножается на `2` (получаем `10`), а в последней операции из **`_`** вычитается `5` (получаем `5`).

### 2. Работа с объектами

```javascript
> let user = { name: "Alice", age: 25 }
undefined
> _.name
"Alice"
> let names = ["Alice", "Bob", "Charlie"]
undefined
> _.length
3
```

После создания объекта `user`, **`_`** хранит этот объект, что позволяет получить доступ к его свойству `name`.
Затем создается массив `names`, и **`_`** уже ссылается на этот массив.

### 3. Вызовы методов и цепочки

```javascript
> "hello world".toUpperCase()
"HELLO WORLD"
> _.split(" ")
[ "HELLO", "WORLD" ]
> _[0]
"HELLO"
```

Сначала **`_`** содержит строку, преобразованную в верхний регистр, затем массив после разделения, а затем первый элемент этого массива.

## Особенности поведения

### Команды, не возвращающие значения

Если команда не возвращает явного значения (например, присваивание), **`_`** не будет обновлена:

```javascript
> let x = 10
undefined
> _
undefined
> 42
42
> _
42
```

Здесь после присваивания **`_`** остается `undefined`, но после вычисления числа `42`, **`_`** обновляется.

### Использование \_ в выражениях

Можно использовать **`_`** непосредственно в выражениях:

```javascript
> 5 * 5
25
> _ + 15
40
> Math.sqrt(_)
6.324555320336759
```

## Практическое применение

Переменная **`_`** особенно полезна при:

- **Интерактивной отладке**: позволяет быстро проверять свойства и методы объектов
- **Пошаговых вычислениях**: избавляет от необходимости создавать множество временных переменных
- **Экспериментах с API**: помогает быстро тестировать функции и их результаты

## Заключение

Переменная **`_`** в Node.js REPL — это удобный инструмент, который упрощает тестирование и взаимодействие с кодом, делая процесс разработки более эффективным и плавным. Она позволяет быстро работать с результатами предыдущих операций, не создавая дополнительных переменных.
