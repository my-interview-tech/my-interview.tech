Объекты `window` и `document` существуют только в **браузере**, но отсутствуют в **Node.js**, потому что:

1. **Node.js – это серверная среда**
    - В отличие от браузера, где `window` представляет глобальный объект страницы, в Node.js нет графического интерфейса и DOM.
    - Вместо `window` в Node.js используется **`global`** как глобальный объект.
      
2. **Отсутствие DOM**
    - `document` управляет HTML-страницей в браузере, но в Node.js нет HTML-документа, так как он предназначен для серверных задач.
      
3. **Разные цели браузера и Node.js**
    - Браузерная среда создана для работы с пользовательским интерфейсом.
    - Node.js ориентирован на **серверные операции** (работу с файлами, сетью, базами данных).
