Чтобы удалить установленный npm-пакет, используйте команду **`npm uninstall`** (или **`npm remove`**, что является синонимом). Существует несколько способов удаления пакетов в зависимости от того, как и где они были установлены.

## 1. Удаление локально установленного пакета

Если пакет был установлен **локально** (в проекте), используйте команду:

```bash
npm uninstall <пакет>
```

### Пример:

```bash
npm uninstall lodash
```

Это удалит пакет `lodash` из папки **`node_modules`** и также удалит запись о нем из **`package.json`** (если это зависимость, а не dev-зависимость).

Если вы хотите удалить пакет, но оставить его запись в **`package.json`**, используйте флаг `--no-save`:

```bash
npm uninstall lodash --no-save
```

## 2. Удаление dev-зависимости

Если пакет был установлен как **dev-зависимость** (devDependency), используйте команду:

```bash
npm uninstall <пакет> --save-dev
```

### Пример:

```bash
npm uninstall jest --save-dev
```

Это удалит пакет из **`node_modules`** и **`devDependencies`** в **`package.json`**.

## 3. Удаление глобально установленного пакета

Если пакет был установлен **глобально** (например, с флагом `-g`), используйте команду:

```bash
npm uninstall -g <пакет>
```

### Пример:

```bash
npm uninstall -g typescript
```

Это удалит глобально установленный пакет `typescript`.

## 4. Удаление нескольких пакетов

Вы также можете удалить несколько пакетов одновременно, перечислив их через пробел:

```bash
npm uninstall <пакет1> <пакет2>
```

### Пример:

```bash
npm uninstall lodash axios
```

## 5. Проверка после удаления

Чтобы убедиться, что пакет был успешно удален:

```bash
# Для локальных пакетов
npm list | grep <имя-пакета>

# Для глобальных пакетов
npm list -g | grep <имя-пакета>
```

## Краткая справка по командам

| Команда                            | Описание                                           |
| ---------------------------------- | -------------------------------------------------- |
| `npm uninstall <пакет>`            | Удаляет локальный пакет и обновляет `package.json` |
| `npm uninstall <пакет> --no-save`  | Удаляет пакет без изменений в `package.json`       |
| `npm uninstall <пакет> --save-dev` | Удаляет пакет из dev-зависимостей                  |
| `npm uninstall -g <пакет>`         | Удаляет глобальный пакет                           |
| `npm rm <пакет>`                   | Синоним для `npm uninstall`                        |
