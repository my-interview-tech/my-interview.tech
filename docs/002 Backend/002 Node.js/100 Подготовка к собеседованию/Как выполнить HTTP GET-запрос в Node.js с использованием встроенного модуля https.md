Для выполнения HTTP GET-запроса в Node.js с использованием встроенного модуля `https`, можно использовать метод `https.get()`. Этот метод автоматически отправляет GET-запрос и возвращает объект ответа. Вот пример, как это сделать:

### Пример:

``const https = require('https');  const url = 'https://www.example.com'; // Укажите нужный URL  https.get(url, (res) => {   let data = '';    // Чтение данных по мере поступления   res.on('data', (chunk) => {     data += chunk;   });    // Когда все данные получены   res.on('end', () => {     console.log('Response Body:', data); // Выводим тело ответа   });  }).on('error', (e) => {   console.error(`Ошибка при запросе: ${e.message}`); // Обработка ошибок });``

### Объяснение:

1. Мы подключаем модуль `https` с помощью `require('https')`.
2. Используем метод `https.get()` для отправки GET-запроса.
    - В аргумент передаем URL, на который нужно отправить запрос.
    - В качестве второго аргумента передаем колбэк-функцию, которая будет вызвана, когда сервер ответит. Этот колбэк получает объект `res`, который представляет собой ответ.
3. Внутри колбэк-функции:
    - Слушаем событие `data`, чтобы собрать данные ответа по мере их поступления.
    - Слушаем событие `end`, чтобы обработать все данные после завершения запроса.
4. В случае ошибки (например, проблемы с сетью) мы ловим событие `error` и выводим сообщение об ошибке.

Этот код выполняет GET-запрос на указанный URL и выводит содержимое ответа в консоль.
