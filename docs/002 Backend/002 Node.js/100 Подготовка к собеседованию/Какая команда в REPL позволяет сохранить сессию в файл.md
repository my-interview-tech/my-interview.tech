В **Node.js REPL** команда **`.save`** позволяет сохранить текущую сессию в файл.

### Синтаксис:

bash

КопироватьРедактировать

`.save <путь к файлу>`

### Пример использования:

1. Откройте REPL с помощью команды:
    
    bash
    
    КопироватьРедактировать
    
    `node`
    
2. Введите какие-либо команды в REPL:
    
    javascript
    
    КопироватьРедактировать
    
    `> const a = 10; > const b = 20; > a + b 30`
    
3. Чтобы сохранить текущую сессию в файл (например, в файл `session.js`), выполните команду:
    
    bash
    
    КопироватьРедактировать
    
    `.save session.js`
    
4. В результате файл `session.js` будет содержать все введенные в REPL команды:
    
    javascript
    
    КопироватьРедактировать
    
    `const a = 10; const b = 20; a + b;`
    

### Примечания:

- Файл будет создан в текущей рабочей директории, если путь к файлу не указан полностью.
- Вы можете указать полный путь к файлу, чтобы сохранить сессию в нужное место.

Эта команда полезна, если вы хотите сохранить результаты работы с REPL и продолжить их использовать в будущем.
