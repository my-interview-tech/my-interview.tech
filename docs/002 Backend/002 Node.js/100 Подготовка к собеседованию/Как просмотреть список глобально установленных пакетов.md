Чтобы просмотреть список **глобально установленных пакетов** в **npm**, можно использовать команду:

### **`npm list -g --depth=0`**

- **`-g`** указывает на то, что вы хотите увидеть глобально установленные пакеты.
- **`--depth=0`** ограничивает вывод только первичными глобальными зависимостями (без вложенных зависимостей).

#### Пример:

```bash
npm list -g --depth=0
```

Этот вывод покажет только глобальные пакеты, установленные непосредственно, без их вложенных зависимостей. Пример вывода:

```bash
/usr/local/lib
├── typescript@4.5.5
├── eslint@7.32.0
├── nodemon@2.0.15
└── yarn@1.22.19
```

### Другие полезные команды для работы с глобальными пакетами:

- **Просмотр глобальных пакетов с вложенными зависимостями:**
  ```bash
  npm list -g
  ```
- **Просмотр глобальных пакетов, включая их версии и пути установки:**
  ```bash
  npm ls -g --json
  ```
- **Просмотр глобальных пакетов без вывода зависимостей (в формате JSON):**
  ```bash
  npm list -g --depth=0 --json
  ```

### Заключение:

- **`npm list -g --depth=0`** — это основной способ просмотреть список глобально установленных пакетов в вашем окружении Node.js.
