Для просмотра всех доступных версий npm-пакета существует несколько методов. Вот наиболее удобные способы:

### 1. Использование команды `npm view`

Самый простой и рекомендуемый способ посмотреть все доступные версии пакета:

```bash
npm view <имя-пакета> versions
```

Пример:

```bash
npm view react versions
```

Вывод будет представлять собой массив доступных версий:

```
[
  '0.3.0',  '0.3.1',  '0.4.0',  '0.4.1',  '0.5.0',
  ...,
  '17.0.0', '17.0.1', '17.0.2', '18.0.0', '18.1.0'
]
```

### 2. Получение полной информации о пакете

Если вы хотите увидеть не только версии, но и другую информацию о пакете:

```bash
npm view <имя-пакета>
```

Эта команда выведет полную информацию о пакете, включая описание, автора, зависимости, и все доступные версии.

### 3. Просмотр конкретной информации с помощью `npm view`

Вы можете получить конкретную информацию о пакете, включая:

```bash
# Только последнюю версию
npm view <имя-пакета> version

# Конкретные свойства (например, зависимости последней версии)
npm view <имя-пакета> dependencies

# Информацию о конкретной версии
npm view <имя-пакета>@<версия>
```

### 4. Использование npm-интерфейса в браузере

Вы также можете просмотреть доступные версии через веб-интерфейс npm:

1. Перейдите на сайт [npmjs.com](https://www.npmjs.com/)
2. Найдите нужный пакет через поиск
3. На странице пакета будет вкладка "Versions", где можно увидеть историю версий

### 5. Использование утилит командной строки

Вы можете использовать команду `npm view` в сочетании с другими CLI-утилитами для более удобного отображения:

```bash
# Сортировка версий с помощью sort
npm view <имя-пакета> versions --json | jq -r '.[]' | sort -V

# Получение только последних 5 версий
npm view <имя-пакета> versions --json | jq -r '.[-5:][]'
```

### 6. Использование API npm-реестра

Для программного доступа можно использовать API npm-реестра:

```bash
curl -s https://registry.npmjs.org/<имя-пакета> | jq '.versions | keys'
```

### Заключение

Наиболее удобным и простым способом просмотра доступных версий npm-пакета является команда `npm view <имя-пакета> versions`. Эта команда предоставляет чистый и структурированный список всех доступных версий пакета, что облегчает выбор нужной версии для установки.
