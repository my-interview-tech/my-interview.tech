В **Node.js REPL** можно загрузить внешний модуль (например, **lodash**) в текущую сессию с помощью команды `require`.

### Шаги для загрузки внешнего модуля:

1. Откройте REPL, запустив команду:
    
    bash
    
    КопироватьРедактировать
    
    `node`
    
2. Используйте команду `require`, чтобы загрузить нужный модуль. Например, чтобы загрузить **lodash**:
    
    javascript
    
    КопироватьРедактировать
    
    `> const _ = require('lodash');`
    
3. После этого вы можете использовать функции из **lodash** в текущей сессии REPL:
    
    javascript
    
    КопироватьРедактировать
    
    `> _.camelCase('hello world') 'helloWorld'`
    

### Заметки:

- Перед этим убедитесь, что модуль (например, **lodash**) установлен в вашем проекте. Если он не установлен, вы можете установить его с помощью npm:
    
    bash
    
    КопироватьРедактировать
    
    `npm install lodash`
    
- Важно, что **REPL** автоматически использует **`require`** для загрузки стандартных модулей Node.js и внешних зависимостей, установленных в вашем проекте (например, через **npm**).
