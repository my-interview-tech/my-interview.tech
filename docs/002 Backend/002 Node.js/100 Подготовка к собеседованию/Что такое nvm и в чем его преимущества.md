**nvm** (Node Version Manager) — это утилита, которая позволяет управлять несколькими версиями Node.js на одной машине. Она особенно полезна для разработчиков, которые работают над несколькими проектами, требующими разных версий Node.js. 

Он позволяет:

- Устанавливать и переключать версии Node.js.
- Работать с несколькими проектами, требующими разных версий Node.js.
- Автоматизировать переключение версий с помощью `.nvmrc`.
