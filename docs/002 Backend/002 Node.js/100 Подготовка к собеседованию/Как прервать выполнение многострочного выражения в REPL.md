В **Node.js REPL** вы можете прервать выполнение многострочного выражения или прекратить ввод многострочного кода с помощью сочетания клавиш.

### Для прерывания ввода многострочного выражения в REPL:

1. **`Ctrl + C`** — это сочетание клавиш, которое можно использовать для прерывания ввода многострочного выражения. Это завершает текущий ввод и возвращает вас к новой строке ввода.

   - Если вы используете многострочное выражение и хотите прекратить его ввод, достаточно нажать **`Ctrl + C`** один раз. Это выведет REPL обратно в основной режим, и больше не будет ожидать ввода следующей строки.

2. **Два раза нажать `Ctrl + C`** — если вы хотите полностью завершить выполнение REPL-сессии, можно нажать **`Ctrl + C`** дважды подряд. Это приведет к выходу из REPL и завершению работы Node.js.

### Пример:

1. Введите многострочное выражение:

   ```javascript
   > if (true) {
   ...   console.log('Hello');
   ... }
   ```

2. Если вы хотите прекратить ввод этого выражения, нажмите **`Ctrl + C`** один раз:

   ```bash
   ^C
   >
   ```

3. Теперь REPL вернется в основной режим и больше не будет ожидать завершения многострочного выражения.

### Примечания:

- Использование **`Ctrl + C`** один раз отменяет только текущий ввод, но не завершает сессию REPL.
- Если вы хотите завершить REPL-сессию, используйте **`Ctrl + C`** дважды.

Эти сочетания клавиш полезны, если вы случайно начали вводить сложную конструкцию и хотите прервать её ввод до того, как она будет выполнена.
