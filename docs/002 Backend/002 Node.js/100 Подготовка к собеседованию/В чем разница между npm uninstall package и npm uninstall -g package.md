Разница между командами **`npm uninstall <пакет>`** и **`npm uninstall -g <пакет>`** заключается в том, где и как удаляется пакет:

### 1. **`npm uninstall <пакет>`** (локальное удаление)

- **Что делает:** Удаляет **локально установленный пакет**, который находится в папке **`node_modules`** в вашем проекте.
- **Где используется:** Используется в контексте проекта, когда вы хотите удалить зависимость, установленную для вашего приложения.
- **Что происходит:**
    - Пакет удаляется из папки **`node_modules`**.
    - Если этот пакет был записан в **`package.json`** (в разделе `dependencies` или `devDependencies`), запись об этом пакете будет удалена.

#### Пример:

bash

КопироватьРедактировать

`npm uninstall lodash`

Удаляет пакет `lodash` из **`node_modules`** и обновляет **`package.json`** вашего проекта.

### 2. **`npm uninstall -g <пакет>`** (глобальное удаление)

- **Что делает:** Удаляет **глобально установленный пакет**, который был установлен с флагом `-g`.
- **Где используется:** Используется для удаления пакетов, установленных глобально для использования в любой директории системы (например, для утилит командной строки).
- **Что происходит:**
    - Пакет удаляется из глобальной папки npm (обычно это `npm` или `node_modules` в системной директории).
    - Он больше не будет доступен для использования через командную строку.

#### Пример:

bash

`npm uninstall -g typescript`

Удаляет глобально установленный пакет `typescript` и делает его недоступным в командной строке.

### Ключевая разница:

- **`npm uninstall <пакет>`** удаляет локальные зависимости, связанные с вашим проектом.
- **`npm uninstall -g <пакет>`** удаляет глобально установленные пакеты, которые можно использовать в любой директории вашего компьютера.

### Пример использования:

- Если вы используете какой-либо инструмент в командной строке, установленный глобально (например, `typescript` или `create-react-app`), и хотите его удалить, используйте **`npm uninstall -g <пакет>`**.
- Если же пакет является частью вашего проекта (например, `lodash` в вашем проекте), используйте **`npm uninstall <пакет>`**.
