MD5 и другие устаревшие алгоритмы хеширования больше не считаются безопасными для хранения паролей:

- MD5 был взломан и уязвим для коллизий

- Скорость хеширования MD5 позволяет эффективно использовать атаки перебором

- Отсутствие "соли" делает использование радужных таблиц эффективным

**Пример уязвимого кода с MD5:**

```javascript

const md5 = require('md5');

const hashedPassword = md5('пароль_пользователя'); // Небезопасно!

```
