Современные рекомендации по хешированию паролей:

- Использование специализированных функций для хеширования паролей (bcrypt, Argon2, PBKDF2)
- Применение уникальной "соли" для каждого пароля
- Настройка стоимости (количества итераций) в зависимости от производительности системы

**Пример безопасного хранения пароля с bcrypt:**

```javascript
const bcrypt = require("bcrypt")

// Хеширование пароля
async function hashPassword(password) {
  const saltRounds = 12 // Повышенное количество итераций
  const salt = await bcrypt.genSalt(saltRounds)
  return await bcrypt.hash(password, salt)
}

// Проверка пароля
async function verifyPassword(password, hashedPassword) {
  return await bcrypt.compare(password, hashedPassword)
}
```
