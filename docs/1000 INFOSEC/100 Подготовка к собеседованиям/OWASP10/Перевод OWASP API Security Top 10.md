Поскольку API-интерфейсы могут разрешать доступ к информации, позволяющей установить личность, обеспечение безопасности таких сервисов имеет первостепенное значение не только для стабильности бизнеса, но и для соблюдения таких регламентов, как GDPR (General Data Protection Regulation). Главные проблемы в списке OWASP связаны с ненадлежащим использованием идентификации или отсутствием четкой стратегии. Учет рекомендаций и наличие достаточных доказательств для подтверждения, имеет решающее значение для обеспечения того, чтобы те, кто запрашивает данные имели надлежащие полномочия.
