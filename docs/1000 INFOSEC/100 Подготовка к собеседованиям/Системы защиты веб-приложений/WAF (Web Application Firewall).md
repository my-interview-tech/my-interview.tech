**WAF** – это решение, которое защищает веб-приложения от различных атак на прикладном уровне, анализируя HTTP-трафик между приложением и интернетом.

**Функции WAF:**

- Блокировка известных атак (XSS, SQL-инъекции, CSRF)
- Проверка входящих запросов на соответствие правилам безопасности
- Защита от DDoS-атак
- Предотвращение сканирования уязвимостей

**Популярные реализации WAF:**

- ModSecurity (открытый исходник)
- Cloudflare WAF
- AWS WAF
- F5 Advanced WAF

**Методы обхода WAF и их предотвращение**

Злоумышленники используют различные техники для обхода WAF:

- Обфускация вредоносных запросов
- Использование альтернативных кодировок
- HTTP Parameter Pollution (HPP)
- Эксплуатация различий в парсинге запросов между WAF и сервером приложения

**Пример обфускированной XSS-атаки, которая может обойти простые WAF:**

```HTML
<img src="x" onerror="&#97;&#108;&#101;&#114;&#116;&#40;&#39;&#88;&#83;&#83;&#39;&#41;">
```

**Защита от обхода WAF:**

- Регулярное обновление правил безопасности
- Использование WAF в сочетании с другими мерами защиты
- Внедрение многоуровневой защиты
