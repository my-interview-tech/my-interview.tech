Деревья являются подтипом более общей структуры данных - связного графа.

Чтобы быть деревом, граф должен удовлетворять двум требованиям:
* **В нем должны отсутствовать циклы ("петли")**
* **Он должен быть связным.** Для любого данного узла графа должен существовать путь к любому другому узлу, причем только один.

Деревья используются для моделирования различных алгоритмических структур данных, таких как:
* обычные бинарные деревья,
* красно-черные деревья,
* B-деревья,
* AB-деревья,
* 2-3 деревья,
* префиксные деревья,
* двоичные кучи (HEAP) и тд.

Дерево принято называть корневым, если при его построении соблюдены следующие правила:
* выбирается в точности одна вершина, которая называется "Корень".
* вершина "Корень" окрашивается цветом, отличным от других вершин;
* создание нижнего слоя, для каждой ячейки дерева зависит от удаленности от корня - чем больше расстояние, тем ниже ячейки.

Общепринятое обозначение деревьев: T

Ниже представлена функция, которая сравнивает два бинарных дерева:

```js
// Функция для создания узла дерева
const createNode = (val, left = null, right = null) => {
  return { val, left, right };
}

// Функция сравнения двух деревьев
const isSameTree = (p, q) => {
  // Оба узла отсутствуют
  if (p === null && q === null) return true;
  
  // Один из узлов отсутствует
  if (p === null || q === null) return false;
  
  // Значения не совпадают
  if (p.val !== q.val) return false;
  
  // Рекурсивная проверка ветвей
  return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
}


const tree1 = {
  val: 1,
  left: { val: 2, left: null, right: null },
  right: { val: 3, left: null, right: null }
};

const tree2 = {
  val: 1,
  left: { val: 2, left: null, right: null },
  right: { val: 4, left: null, right: null } // Измененное значение
};

console.log(isSameTree(tree1, tree2)); //false


```
