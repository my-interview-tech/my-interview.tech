### Сборщики и модули

##### Webpack

```javascript
// webpack.config.js
module.exports = {
    entry: './src/index.js',
    output: {
        filename: 'bundle.js',
        library: 'MyLibrary',
        libraryTarget: 'umd' // UMD для совместимости
    },
    module: {
        rules: [
            {
                test: /\.js$/,
                use: 'babel-loader'
            }
        ]
    }
};
```

##### Vite/Rollup (ESM-first)

```javascript
// vite.config.js
export default {
    build: {
        lib: {
            entry: './src/index.js',
            formats: ['es', 'cjs', 'umd']
        },
        rollupOptions: {
            external: ['react', 'react-dom']
        }
    }
};
```
