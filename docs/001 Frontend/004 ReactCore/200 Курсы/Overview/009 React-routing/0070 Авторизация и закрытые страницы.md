Можно использовать компонент `Redirect` , чтобы переслать пользователя на логин-страницу :

```jsx
<Redirect to="/login" />
```

Система авторизации , которую мы рассмотрели , не обеспечивает безопасность в приложении (_проверка прав должна проводиться на сервере_)
