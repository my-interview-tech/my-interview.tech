Хук - это обычная функция, имя которой начинается с use.

```jsx
const usePlanetName = (id) => {
	const [ name, setName ] = useState(null);

	useEffect(() => {
		fetch(`/api/planets/${id}/`);
			.then(res => res.json());
			.then(data => setName(data.name));
	}, [id]);

	return name;
}
```
